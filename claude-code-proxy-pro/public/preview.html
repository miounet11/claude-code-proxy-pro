✨ 主要改进

  • 侧边栏导航：采用固定侧边栏设计，配置文件一目了然
  • 多配置管理：支持创建和管理最多10个不同的API配置

  • 模型选择优化：预设主流大小模型，支持自定义输入
  • 视觉优化：更现代的配色方案和交互动画
  • 状态指示：清晰的在线/离线状态显示
  • 快速操作：集成终端打开、脚本导出等快捷功能

🎯 用户体验提升

  • 配置切换更便捷，点击即可切换不同配置
  • 表单分组更清晰，基础设置、API设置、模型设置分开展示
  • 实时保存提示，操作反馈更及时
  • 空状态引导，新用户上手更容易

  • 优化的动画性能，界面更流畅
