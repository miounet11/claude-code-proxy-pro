故障排查

1. 代理是否在运行？

curl -sS http://127.0.0.1:8082/health | jq .

若失败，点击应用右下角“状态面板 → 一键修复”。

2. 环境变量是否生效？

echo $ANTHROPIC_BASE_URL  # Windows PowerShell: $env:ANTHROPIC_BASE_URL

若未指向 127.0.0.1:8082，执行：

bash scripts/configure-claude-code.sh --anthropic-key my-proxy-key --port 8082 --persist

3. 端口被占用？

lsof -iTCP -sTCP:LISTEN -P -n | grep :8082  # macOS/Linux
netstat -ano | findstr :8082                # Windows

应用会自动尝试在 8082-8102 选择可用端口，或在“安装脚本”中指定 --port。

4. 系统代理/公司网络影响？

确保 NO_PROXY/no_proxy 包含 localhost,127.0.0.1（应用已自动合并），重启 VS Code/终端后重试。

导出诊断包

应用内“系统体检 → 导出诊断包”并发送给支持人员（Key 已脱敏）。
